> 2048 게임을 **Vanilla JavaScript** 환경에서 직접 구현한 웹 게임 프로젝트입니다.

#### 🎯 프로젝트 소개

> **기존의 2048 게임을 저희 팀만의 로직과 디자인으로 재해석한 프로젝트입니다.**  
> 코드의 간결함과 유지 보수성을 높이는 데 집중하였습니다.
